Make Sale

Make Sale

Purpose

The purpose of this function is to allow the user to record a sale that has been made on the system. This function records details of the sale and of the customer that wants to purchase goods.


How it Works?

The user will enter the name of the customer in the search bar found on the right-hand side of the screen. If it is a new customer, the user will click the        button and proceed to add the customer (see pg. 2). Once the customer has been selected, the user will use the search bar on the left side of the screen to search for the product the customer would like to purchase. Once the product has been selected, it will appear in the table below. To remove a product from the sale, the user must click the    button next to the total of the product. If the customer would like to have the sale delivered to them, the user must select the         checkbox. Once all the products have been added, the user must click the        button at the bottom of the screen. The sales manager is then requested to enter his password to approve the sale. This will finalise the sale and print an invoice for the customer.
