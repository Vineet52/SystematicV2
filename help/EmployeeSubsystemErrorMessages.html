Employee Subsystem Error Messages


Employee Subsystem Errors  



Only JPEG images can be uploaded. Please check file uploads highlighted in red.

This error describes an event in which an image of the wrong file format was upload, the system only allows for JPEG format images to be saved to the server.

To resolve this error click the close button in the file input upload box and select a file of type JPEG to upload.

Only Images less than 20MB can be uploaded. Please check file uploads highlighted in red

This error describes an event where the file you have selected to upload is larger than 20MB , the system does not allow users to upload Image files that are larger than 20MB for storage space reasons.

To resolve this error click close and select a image less than 20MB to upload.

Employee exists! , press close and try again

If the details entered match an existing employee, E.g Identification number input the system will not allow the employee to be registered.

To resolve this error click close and check your inputs captured to make sure that they are unique , if this error persists contact the system administrator who will search for the employee details you have entered.

Error saving employee picture or generated employee tag

This is a rare error that checks for typing error in your files uploaded to prevent SQL injections and malicious files , please make sure your files are a valid type when adding an employee.

To resolve this error click the close button and please make sure your files are a valid type when adding an employee.







Database error

This error refers to a database timeout error where the attempt to connect to the database has failed due to a connection error.

To resolve this error one must refresh their page then proceed to retry , making sure they have a strong and steady internet connection as the database is cloud based.

If this error persist contact the system administrator.

Employee QR codes could not be generated


This error refers to a QR error whereby the connection to the online QR CDN has failed due to a timeout.


To resolve this error one must refresh their page then proceed to retry , making sure they have a strong and steady internet connection as the database is cloud based.

If this error persist contact the system administrator.

Employee does not earn wage, thus an employee tag is not generated

This error refers to a user trying to generate an employee tag for an employee who does not earn a wage , thus they do not have an employee tag.

To resolve this error click the close button , if employee is wage earning contact the system administrator.


Already Checked-in!

This error describes an event where the user tries to check in an employee who is already checked-in to the system.

Click close to resolve this error.


Check-in first


This error refers to where the user tries to check out an employee who is not checked-in to the system.

To resolve this error and successfully checkout the employee check him/her first into the system.
