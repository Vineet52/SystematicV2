Add Supplier


Purpose

This function allows the user to add a supplier to the system. The user enters the supplier details and they are saved on the system.


How it Works?

The user enters the applicable details into the input box on the screen. When the supplier details have been entered fully and correctly, the user must click the               button to save the new supplier details onto the system.

Screen Interface











Detailed Explanation of the Screen

Number

Description

1

5

Allows user to enter an address of the supplier. This field uses autocomplete functionality – the user may start typing their address and the system will display results for the user to select. If the address does not appear, it will be added as a new address.

6

This field will be automatically filled out after the address is autocompleted. If the address is not found with autocomplete, the user enters this manually.

This button must be clicked if a customer has another address they would like to add.
