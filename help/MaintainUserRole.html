Maintain User Role

Maintain User Role

Purpose

This function allows a specified user role to be maintained, should the user role details need to be updated or the functionalities need to be changed.


How it Works

The user role details may then be changed by entering new details into the input boxes provided and selecting the desired functionalities from the drop-down list. Once the details have been updated, the button must be clicked for the new user role details to be saved onto the system.

 

The user role may be deleted by clicking the button found on the right-hand corner of the screen.


Screen Interface

Detailed Explanation of the Screen

Number

Description

1

Allows user to enter a position name of the user role to be updated.

2

Allows user to select a user role function for the specific position name.

3

Allows user to save the updated user role onto the system.

4

Allows user to delete the user role from the system
