Maintain Employee

Maintain Employee

Purpose

This system function is used to update employee details, should any details of an existing employee need to be updated.


How it Works

The user will then enter any details that need to be changed and click the          button – the updated employee details are saved on the system.


Screen Interface





Detailed Explanation of the Screen

Number

Description

1

Allows user to enter an employee name.

2

Allows user to enter a surname.

3

Allows user to select a title from the drop-down list.

4

Allows user to enter contact number or select values to represent the contact number.

5

Allows user to enter email address for the employee.

6

Allows user to enter SA ID/Passport number or select values to represent SA ID/Passport number.

7

Allows user to enter an address of the employee. This field uses autocomplete functionality – the user may start typing their address and the system will display results for the user to select. If the address does not appear, it will be added as a new address.

8

This field will be automatically filled out after the address is autocompleted. If the address is not found with autocomplete, the user enters this manually.

9

This field will be automatically filled out after the address is autocompleted. If the address is not found with autocomplete, the user enters this manually.

10

This field will be automatically filled out after the address is autocompleted. If the address is not found with autocomplete, the user enters this manually.

11

Allows user to select an employee type that the employee is categorised as.

12

This button will open a terminal to choose a file from the PC to be uploaded as an employee picture.

13

This button saves the employee details to the system.
