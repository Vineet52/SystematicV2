Register Customer

Register customer 

Purpose

This system function is used to register new customer on the system. The interactive form has two tabs, for an individual and organisational customer to register by capturing their details.

Screen interface – Individual



Detailed Explanation of the Screen - Individual


Number

Description

1

Allows the user to enter details for a customer of type individual

2

Allows the user to enter details for a customer of type organisation.

3

Enter the customer’s first name.

4

Enter the customer’s surname.

5

Allows user to select a title.

6

Allows user to enter contact number.

7

Allows user to enter email address.

8

Allows user to enter an address of the customer. This field uses autocomplete functionality – the user may start typing their address and the system will display results for the user to select. If the address does not appear, it will be added as a new address.

9

This field will be automatically filled out after the address is autocompleted. If the address is not found with autocomplete, the user enters this manually.

10

This field will be automatically filled out after the address is autocompleted. If the address is not found with autocomplete, the user enters this manually.

11

This field will be automatically filled out after the address is autocompleted. If the address is not found with autocomplete, the user enters this manually.

12

This button must be clicked if a customer has another address they would like to add.

13

Allows the user to save the individuals details onto the system.
