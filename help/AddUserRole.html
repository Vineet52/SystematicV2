Add User Role

Purpose

This function allows a new user role to be added. Different user roles may be created to allow for specific access of the Stock Path system. For example, some cashiers may not be allowed to access delivery / collection functionalities, so a user role will restrict their functionality accordingly.


How it Works



The position name will be entered, and the functionalities must be selected from the drop-down list. Finally, the button must be clicked for the details to be saved on the system.

Screen Interface



Detailed Explanation of the Screen


Number

Description

1

Allows user to enter a position name of the user role to be added.

2

Allows user to select a user role function for the specific position name.

3

Allows user to save the newly added user role onto the system.
