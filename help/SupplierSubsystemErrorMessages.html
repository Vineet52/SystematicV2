This error is shown when the user has not added the supplier name to which he/she is trying to order from.


This error is resolved by adding a supplier name, to which the order is being made.





This error is shown when one of the input quantities are zero.

This error can be resolved by inputting a quantity higher than zero in the highlighted quantity box.


This error is shown when no quantities are more than zero on the screen.

This error can be resolved by inputting all quantities for each quantity box of which the quantity value is greater than zero in the highlighted quantity box.
