Product Subsystem Error Messages


Product Subsystem

Errors

No changes made to product error message

This error message appears when there have been no changes made to the product.

       

To resolve this error message, enter changes to the product. Click the ‘close’ button to close the modal.


A Product with the changed name already exists

This error message occurs when there is a product on the system that has the same name as the product to be maintained.

To resolve the error, ensure the name of the product to be maintained is not the same as an existing product. Click the ‘Close’ button to close the modal.

Database Error

This error occurs in the event where a database error has occurred due to a failed connection to the server.

To resolve this error, refresh the page and try submitting again and make sure to have a stable internet connection.




A Product type with the same name already exists

This error occurs when there is an existing product type on the system that has the same name as the one the user is attempting to add.

To resolve this error, try adding a product type with a name that is not the same as the name of a product type that already exists on the system.

This product type already exists

This error occurs when there is an existing product type on the system that has the same details as the one the user is attempting to add.

To resolve this error, enter product type details that do not match details of an existing product type.


Error Adding Product Type

This error occurs when an error has occurred when attempting to add a product type.

To resolve this error, try entering the product type again. If the problem persists, contact the systems administrator.

Error adding product

This error occurs when an error has occurred when attempting to add a product.


To resolve this error, try entering the product again. If the problem persists, contact the systems administrator.




This product type already exists

This error occurs when the user attempts to add a product when a product with the same details already exists on the system.

To resolve this error, try adding a product that does not have the same details as one that already exists on the system.
