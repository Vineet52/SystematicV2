Search User Role

Search User Role

Purpose

This function allows for a specific user role to be searched.


How it Works?

The user role name must be entered into the search bar that is displayed on the screen. The system dynamically searches for the matching user role and displays the desired results.


Screen Interface










Detailed Explanation of the Screen

Number

Description

1

Allows the user to enter the user role name.

2

Redirects user to maintain user role screen.

3

Redirects the user to the user landing page.
